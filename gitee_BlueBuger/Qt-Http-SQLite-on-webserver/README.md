# Qt Http SQLite on webserver

#### 介绍
使用qt 的 json 和 sql类；
使用mongoose httpserver
使用 bootstrap；

该demo能将本地的sqlite数据库放到 浏览器中打开，查询，能够在浏览器中运行sql语句

html前端部分 来自于 https://github.com/skyhacker2/SQLiteOnWeb-Android
